วิธีเล่นแบบใหม่ที่เข้าใจง่ายและเป็นมิตรยิ่งขึ้น 🎮
เมื่อนำไฟล์ไปรันบนเว็บเบราว์เซอร์ให้ปฏิบัติตามขั้นตอนด้านล่างได้ทันทีครับ:

เตรียมตำแหน่งยืน: ยืนห่างจากหน้าจอเล็กน้อยให้กล้องจับภาพมือและข้อมือได้อย่างชัดเจน (ระบบจะเริ่มตรวจจับเมื่อคุณชูมือขึ้นมาในเฟรมกล้อง)

เล็งตำแหน่งแตะ: * คุณจะสังเกตเห็น "เป้าเล็งสีฟ้า" (Pointer) ที่ติดอยู่บนปลายนิ้วชี้ของคุณเสมือนเมาส์

เลื่อนมือของคุณให้ "เป้าสีฟ้า" นั้นไปชี้ทับแผ่นป้ายกระดูกที่อยู่แถบซ้ายหรือขวา

ทันทีที่เป้าแตะโดน แผ่นป้ายนั้นจะเปลี่ยนเป็น "สีชมพูแดงเรืองแสง" ทันที เพื่อบอกใบ้ให้คุณรู้ว่า "ชี้โดนป้ายนี้เรียบร้อยแล้ว!"

คีบหยิบและลาก:

ให้คุณ "จีบนิ้วชี้และนิ้วโป้งเข้าหากัน" ค้างไว้ (เหมือนท่าคีบหยิบสิ่งของทั่วไป)

แผ่นป้ายจะเปลี่ยนเป็น "สีน้ำเงินสว่าง" และเคลื่อนย้ายตามนิ้วชี้ของคุณไปรอบหน้าจอ

จัดวางลงข้อต่อกระดูก:

ลากป้ายคำศัพท์นั้นไปทับ "เป้ากากบาท ⊕" สีฟ้าบนตัวโครงกระดูกสีขาวตรงใจกลางจอให้ตรงตำแหน่ง (เช่น ป้ายกะโหลกศีรษะให้เอาไปทับเป้าบริเวณส่วนหัว)

เมื่อจุดอ้างอิงตรงกันแล้ว ให้ "ปล่อยนิ้วโป้งออกจากนิ้วชี้" (เลิกจีบ)

ป้ายคำศัพท์จะถูกดูดเข้าไปติดแน่นอย่างสวยงาม เปลี่ยนสถานะเป็น "สีเขียว" พร้อมมีคำบรรยายขึ้นบอกทางด้านขวาของหน้าจออย่างสมบูรณ์แบบครับ!
